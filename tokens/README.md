# Clickio Design System

Полная копия дизайн-системы test-project для платформы Clickio.

## 📁 Структура проекта

```
clickio/
├── $metadata.json         # Порядок токенов
├── $themes.json          # Темы оформления
├── common/               # Общие токены
│   ├── effect/          # Эффекты и тени
│   ├── layout/          # Лейаут и отступы
│   ├── palette/         # Цветовые палитры
│   └── typography/      # Типографика
├── component/           # Компонентные токены
│   ├── button.json      # Кнопки
│   ├── card.json        # Карточки
│   ├── checkbox.json    # Чекбоксы
│   ├── dropdown.json    # Выпадающие списки
│   ├── form.json        # Формы
│   ├── header.json      # Шапка
│   ├── icon.json        # Иконки
│   ├── input.json       # Поля ввода
│   ├── link.json        # Ссылки
│   ├── modal.json       # Модальные окна
│   ├── pagination.json  # Пагинация
│   ├── progress.json    # Прогресс-бары
│   ├── radio.json       # Радио-кнопки
│   ├── sidebar.json     # Сайдбар
│   ├── switch.json      # Переключатели
│   ├── tabs.json        # Табы
│   └── tooltip.json     # Тултипы
├── figma-only/         # Figma-специфичные токены
└── shared/             # Общие компоненты
```

## 🎨 Цветовая система

Основные цветовые палитры:
- **brand** - основные брендовые цвета
- **content** - цвета для контента
- **dust** - дополнительные цвета
- **stardust** - нейтральные цвета
- **salmon** - акцентные цвета
- **avocado** - цвета для успеха
- **peach** - цвета для предупреждений

## 🧩 Компоненты

Полный набор UI компонентов с токенами:
- Кнопки с различными вариантами и состояниями
- Карточки для разных типов контента
- Формы и поля ввода
- Навигационные элементы
- Модальные окна и оверлеи

## 🚀 Использование

Токены организованы по иерархической структуре:
1. **Common** - базовые токены (цвета, типографика, лейаут)
2. **Component** - составные токены для компонентов
3. **Shared** - общие компоненты и утилиты
